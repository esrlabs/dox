Colored TablesÔÉÅ

Use a list table with class colored and add containers for every cell which shall have a color.
Optionally specify spaced for a cell margin or centered to align the text horizontally in the middle:
.. list-table::
    :class: colored [spaced] [centered]

    * - .. container:: <color>-cell

            <text>
      - .. container:: <color>-cell

            <text>

Color example:

.. list-table::
   :header-rows: 1
   :class: colored centered spaced
