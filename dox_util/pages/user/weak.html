weak

Sphinx is based on docutils. In docutils it’s up to the user to decide whether a missing label shall result in a warning or not. Sphinx does not support this feature.

This extension provides references which behave like regular references but by default missing labels are ignored:

To globally re-enable the warnings, add the following line to conf.py:

enable_weak_warnings = True
